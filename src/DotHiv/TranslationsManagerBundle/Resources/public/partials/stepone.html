1. Choose the Translations

Choose the Google Spreadsheet file you want to use as a new source for translations on the dotHIV platform. Any spreadsheet file is acceptable; however the second (B) column must contain the keys followed by the German translation (C column) and the English translation (D column). Everything else, including formatting, does not matter. Formatting like bold fonts or font size won't be recognized.
